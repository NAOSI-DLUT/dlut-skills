# 检索流程

## 找校园生活信息

1. 先在 DUTManual 站内搜索关键词，如宿舍、校园网、选课、考试、报销、校历。
2. 若涉及政策、时间、费用、账号权限，再找学校对应官方部门页面交叉核验。
3. 输出时区分“手册经验”和“官方规则”。

## 找课程资料

1. 先确认课程中文名、开课学院、教师、学期或资料类型。
2. 在 DUTManual 课程相关页面和 `pan.naosi.org` 跳转页中查资料云盘组织方式。
3. 如资料在大工云盘群组中，提示用户使用校园账号登录并按群组名/课程名搜索申请。
4. 整理结果时给出：入口、搜索关键词、可能的目录名、文件整理建议。

## 在大工云盘内搜索

如需直接调用大工云盘 API（获取 token、列出目录、下载文件），请使用 `dlut-pan` skill。

若用户希望在浏览器中手动操作：
1. 打开 `http://pan.dlut.edu.cn/`，不要默认使用 HTTPS。
2. 点击“登录”进入学校统一身份认证，由用户自行完成登录。
3. 登录后进入 `/disk`，使用页面 UI 搜索文件或群组。

结果整理字段建议：名称、类型、所在空间/群组、路径、大小、更新时间、是否目录。

## 整理用户给出的资料

- 按“课程-年份-教师-章节-资料类型”归档。
- 对试卷、答案、课件、实验手册分别标注来源和适用年份。
- 不把旧资料默认当成最新考试范围；复习计划必须以教师本学期要求为准。
