# 来源地图

## 公开入口

- DUTManual：`https://man.naosi.org/`，大工生存手册，一站式校园生活指南。页面显示由 NAOSI 发起，内容使用 CC BY-NC-SA 4.0 许可；使用时仍需查看页面底部最近更新时间。
- NAOSI：`https://naosi.org/`，网络与开源协会相关入口。
- DUTManual GitHub：从 DUTManual 页面顶部 GitHub 链接进入，用于确认文档源码、贡献方式和最近变更。
- NAOSI 资料云盘：优先访问 `https://pan.naosi.org/`，若跳转到 DUTManual 资料页，按跳转后的页面为准。
- 大连理工大学云盘：优先使用 `http://pan.dlut.edu.cn/`。实测 `https://pan.dlut.edu.cn/` 可能连接失败或证书/连接异常；HTTP 首页可打开，标题为“大工云盘”。
- 网信中心云盘服务页：可搜索“网络与信息化中心 云盘服务 大连理工大学”，用于核验大工云盘用途、客户端和群组功能。

## 使用原则

- 公开页面可以总结；登录后目录、分享内容和群组资料只在用户提供材料时整理。
- 对资料云盘只描述“如何找”和“如何整理”，不复制未授权课程资料正文。
- 涉及账号登录、门户认证、客户端版本、群组权限时，要求用户以当前页面为准。
- 如用户希望 agent 帮忙搜索大工云盘，必须让用户自行登录，并在每次搜索前确认只读取搜索结果和元数据。

## 常用核验问题

- 链接是否仍可访问，是否发生跳转。
- 页面底部或仓库提交是否有最近更新时间。
- 资料是否公开授权，还是仅限课程/群组成员。
- 用户需要的是入口、检索关键词、资料整理，还是具体课程复习建议。
