# 数据库系统课程索引

## 章节与知识点

1. **绪论**
   - 数据、数据库、DBMS、DBS 概念
   - 数据模型：概念模型（E-R）、逻辑模型（关系/层次/网状）、物理模型
   - 数据库三级模式与两级映像
2. **关系数据库**
   - 关系数据结构、关系操作（选择、投影、连接、除）
   - 关系代数与关系演算
   - 完整性约束：实体、参照、用户定义
3. **SQL**
   - DDL（CREATE/ALTER/DROP）、DML（SELECT/INSERT/UPDATE/DELETE）
   - 复杂查询：嵌套查询、连接查询、集合查询
   - 视图、索引、触发器、存储过程
4. **关系数据库理论**
   - 函数依赖、Armstrong 公理
   - 范式：1NF、2NF、3NF、BCNF
   - 模式分解：无损连接、保持依赖
5. **数据库设计**
   - 需求分析、概念设计（E-R 图）、逻辑设计、物理设计
   - E-R 图向关系模型的转换
6. **事务与并发**
   - ACID 特性
   - 并发控制：封锁、死锁、活锁
   - 隔离级别：RU、RC、RR、Serializable
   - 故障恢复：日志、检查点、备份
7. **数据库新技术**
   - NoSQL、分布式数据库、云数据库
