# 数据库系统实验工具

## MySQL
- 安装：官网下载或使用 XAMPP/WAMP
- 命令行：`mysql -u root -p`
- 常用命令：`SHOW DATABASES; USE db; SHOW TABLES; DESC table;`

## PostgreSQL
- 安装：官网下载或使用 Docker
- 命令行：`psql -U postgres`
- GUI 工具：pgAdmin、DBeaver

## 可视化工具
- Navicat、DataGrip、MySQL Workbench
- 用于设计 E-R 图、执行 SQL、导入导出数据

## Python 连接
```python
import pymysql
# 或
import psycopg2
```
