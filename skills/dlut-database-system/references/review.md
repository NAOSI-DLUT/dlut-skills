# 数据库系统复习要点

## 核心概念对比
| 概念 | 要点 |
|------|------|
| 关系代数 vs SQL | 关系代数是理论基础，SQL 是实际语言 |
| 内连接 vs 外连接 | 内连接只保留匹配行，外连接保留左/右/全部表行 |
| 视图 vs 基本表 | 视图是虚表，不存储数据；基本表是实表 |

## 易错点
- 自然连接会自动按同名属性等值连接并去重列
- 除运算要找"包含全部"的语义
- 范式分解可能无法同时保持无损连接和函数依赖
- 可串行化调度是并发控制的正确性标准

## 大题常见类型
1. 关系代数表达式与 SQL 互转
2. E-R 图设计与向关系模型转换
3. 函数依赖分析与范式判断/分解
4. 并发调度判断冲突/视图可串行性
5. 日志分析与故障恢复（UNDO/REDO）
