# Java 程序设计工具

## JDK
- 下载：https://www.oracle.com/java/technologies/downloads/
- 环境变量：JAVA_HOME、PATH 添加 %JAVA_HOME%\bin
- 验证：`java -version`、`javac -version`

## IDE
- IntelliJ IDEA（推荐）：Community 版免费
- Eclipse：经典开源选择
- VS Code：配合 Extension Pack for Java

## 构建工具
- Maven：`pom.xml` 管理依赖
- Gradle：`build.gradle` 更灵活

## 调试
- IDE 断点调试
- 命令行：`jdb`
- 日志：SLF4J + Logback

## 在线练习
- LeetCode（Java 模式）
- 牛客网 Java 题库
- HackerRank Java
