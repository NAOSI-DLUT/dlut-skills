# Java 程序设计课程索引

## 章节与知识点

1. **Java 基础**
   - 数据类型、运算符、控制结构、数组
   - 面向对象：类、对象、封装、继承、多态
   - 抽象类、接口、内部类
   - 异常处理：try-catch-finally、throw/throws
2. **常用类库**
   - String/StringBuilder/StringBuffer
   - 包装类、Math、Date/Calendar
   - 泛型：泛型类、泛型方法、通配符
3. **集合框架**
   - List：ArrayList、LinkedList
   - Set：HashSet、TreeSet
   - Map：HashMap、TreeMap
   - 迭代器、比较器 Comparator
4. **IO 与 NIO**
   - 字节流/字符流：InputStream/OutputStream、Reader/Writer
   - 文件操作：File、Files
   - 序列化/反序列化
5. **多线程**
   - Thread、Runnable、Callable
   - 线程同步：synchronized、Lock、ReentrantLock
   - 线程池：ExecutorService
   - 并发集合、原子类
6. **网络编程**
   - Socket、ServerSocket
   - URL、URLConnection
   - NIO 网络编程基础
7. **JDBC**
   - 驱动加载、连接数据库
   - Statement、PreparedStatement、ResultSet
   - 事务处理
8. **GUI（选修）**
   - Swing：JFrame、JPanel、事件监听
   - JavaFX 基础
