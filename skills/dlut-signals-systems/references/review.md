# 信号与系统复习要点

## 核心概念对比
| 概念 | 要点 |
|------|------|
| 能量信号 vs 功率信号 | 能量有限/无限，功率有限/无限；周期信号是功率信号 |
| 因果 vs 稳定 | 因果：h(t)=0, t<0；稳定：∫|h(t)|dt < ∞ |
| FT vs LT vs ZT | FT 分析频谱，LT 分析连续系统，ZT 分析离散系统 |

## 易错点
- 冲激函数 δ(t) 的筛选性质：∫x(t)δ(t-t₀)dt = x(t₀)
- 卷积积分的积分限由信号的非零区间决定
- 傅里叶变换存在条件（狄利克雷条件）比拉普拉斯更严格
- Z 变换收敛域是圆环，极点决定边界

## 大题常见类型
1. 系统性质判断（线性、时不变、因果、稳定）
2. 卷积积分/卷积和计算
3. 傅里叶级数展开（指数形式/三角形式）
4. 傅里叶/拉普拉斯/Z 变换的正反变换
5. 系统函数 H(s)/H(z) 与零极点分析
6. 采样定理应用与频谱分析
