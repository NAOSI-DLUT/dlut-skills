# 信号与系统课程索引

## 章节与知识点

1. **信号与系统概述**
   - 信号分类：连续/离散、周期/非周期、能量/功率
   - 基本信号：单位阶跃、单位冲激、复指数、正弦
   - 系统分类：线性/非线性、时不变/时变、因果/非因果、稳定/不稳定
2. **连续时间系统的时域分析**
   - 微分方程描述
   - 零输入响应、零状态响应
   - 单位冲激响应 h(t)
   - 卷积积分：y(t) = x(t) * h(t)
3. **离散时间系统的时域分析**
   - 差分方程描述
   - 单位脉冲响应 h[n]
   - 卷积和：y[n] = x[n] * h[n]
4. **连续时间信号的频域分析**
   - 傅里叶级数（周期信号）
   - 傅里叶变换（非周期信号）
   - 常用变换对和性质
5. **连续时间系统的频域分析**
   - 系统频率响应 H(jω)
   - 无失真传输、理想滤波器
6. **拉普拉斯变换**
   - 定义、收敛域
   - 常用变换对和性质
   - 系统函数 H(s)
   - 零极点分析、稳定性判断
7. **Z 变换**
   - 定义、收敛域
   - 常用变换对和性质
   - 系统函数 H(z)
   - 零极点分析、稳定性判断
8. **采样定理**
   - 时域采样、频域周期延拓
   - 奈奎斯特采样定理
   - 重建与内插
