# 概率论与数理统计课程索引

## 章节与知识点

1. **概率论基础**
   - 样本空间、随机事件、事件关系与运算
   - 概率定义：古典概型、几何概型、频率、公理化定义
   - 条件概率、全概率公式、贝叶斯公式
   - 事件独立性
2. **随机变量及其分布**
   - 离散型：0-1 分布、二项分布、泊松分布、几何分布
   - 连续型：均匀分布、指数分布、正态分布
   - 分布函数、概率密度函数、概率质量函数
3. **多维随机变量**
   - 联合分布、边缘分布、条件分布
   - 随机变量的独立性
   - 二维正态分布
4. **数字特征**
   - 期望、方差、标准差
   - 协方差、相关系数
   - 矩、协方差矩阵
5. **大数定律与中心极限定理**
   - 切比雪夫不等式
   - 大数定律：切比雪夫、伯努利、辛钦
   - 中心极限定理：独立同分布、棣莫弗-拉普拉斯
6. **数理统计基础**
   - 总体、样本、统计量
   - 三大分布：χ²、t、F
   - 正态总体下的抽样分布
7. **参数估计**
   - 点估计：矩估计、最大似然估计
   - 估计量的评价：无偏性、有效性、一致性
   - 区间估计
8. **假设检验**
   - 基本原理、显著性水平、拒绝域
   - 正态总体参数检验
   - 拟合优度检验（χ² 检验）
9. **回归分析**
   - 一元线性回归：最小二乘估计
   - 显著性检验、预测区间
