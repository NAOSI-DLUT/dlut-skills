# 概率论与数理统计速查

## 常用分布
| 分布 | 记号 | 期望 | 方差 |
|------|------|------|------|
| 0-1 分布 | B(1,p) | p | p(1-p) |
| 二项分布 | B(n,p) | np | np(1-p) |
| 泊松分布 | P(λ) | λ | λ |
| 几何分布 | G(p) | 1/p | (1-p)/p² |
| 均匀分布 | U(a,b) | (a+b)/2 | (b-a)²/12 |
| 指数分布 | Exp(λ) | 1/λ | 1/λ² |
| 正态分布 | N(μ,σ²) | μ | σ² |

## 重要公式
- 全概率：P(A) = Σ P(Bᵢ)P(A|Bᵢ)
- 贝叶斯：P(Bᵢ|A) = P(Bᵢ)P(A|Bᵢ) / Σ P(Bⱼ)P(A|Bⱼ)
- 切比雪夫：P(|X-μ| ≥ ε) ≤ σ²/ε²
- D(X±Y) = D(X) + D(Y) ± 2Cov(X,Y)

## 统计量分布
- χ²(n)：n 个独立标准正态的平方和
- t(n)：Z / √(χ²(n)/n)
- F(m,n)：(χ²(m)/m) / (χ²(n)/n)
