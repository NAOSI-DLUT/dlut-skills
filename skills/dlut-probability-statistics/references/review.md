# 概率论与数理统计复习要点

## 核心概念对比
| 概念 | 要点 |
|------|------|
| 概率 vs 频率 | 概率是理论值，频率是试验值的极限 |
| 互斥 vs 独立 | 互斥：P(AB)=0；独立：P(AB)=P(A)P(B) |
| 无偏 vs 有效 | 无偏：E(θ̂)=θ；有效：方差更小 |

## 易错点
- 泊松分布是二项分布的稀有事件极限（n大p小）
- 指数分布具有无记忆性：P(X>s+t|X>s)=P(X>t)
- 最大似然估计取对数后求导，注意定义域边界
- 假设检验中第一类错误（弃真）概率 = α

## 大题常见类型
1. 条件概率与全概率/贝叶斯公式应用
2. 随机变量函数的分布（分布函数法/公式法）
3. 二维随机变量的联合/边缘/条件分布
4. 矩估计与最大似然估计
5. 正态总体均值/方差的区间估计与假设检验
6. 一元线性回归的最小二乘估计与显著性检验
