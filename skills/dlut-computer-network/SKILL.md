# DLUT Computer Network

## 目标

帮助用户理解计算机网络协议原理、完成网络实验（Socket/Wireshark）、检查配置和计算、整理协议对比和复习。

## 工作流

1. 识别主题：体系结构、物理层、数据链路层、网络层、传输层、应用层或实验。
2. 课程索引读取 `references/course-index.md`。
3. 实验工具和环境读取 `references/tooling.md`。
4. 协议/算法速查读取 `references/cheatsheet.md`。
5. 报告和复习读取 `references/review.md`。

## 质量规则

- 协议分析要分层说明，先定位层再分析字段和行为。
- 子网划分、路由计算要给出步骤和验证方法。
- Socket 代码要检查资源关闭、错误处理和协议匹配。
- Wireshark 过滤表达式要给出可复用的显示/捕获过滤示例。
