# 计算机网络复习要点

## 核心概念对比
| 概念 | 要点 |
|------|------|
| OSI vs TCP/IP | OSI 七层更理论化，TCP/IP 四层更实用 |
| UDP vs TCP | 无连接/面向连接、不可靠/可靠、开销小/大 |
| IPv4 vs IPv6 | 32位/128位、NAT 必要性、首部简化 |

## 易错点
- 子网划分时注意网络地址和广播地址不可分配
- TCP 的滑动窗口是字节为单位，不是报文段
- 三次握手防止历史连接，四次挥手因全双工需双方分别关闭

## 大题常见类型
1. 子网划分与路由表设计
2. TCP 拥塞窗口变化过程绘图
3. CSMA/CD 冲突检测与退避计算
4. Socket 编程代码填空/改错
