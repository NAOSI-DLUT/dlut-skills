# 计算机网络课程索引

## 章节与知识点

1. **概述**
   - 互联网组成、性能指标（速率、带宽、吞吐量、时延、RTT）
   - OSI 七层模型与 TCP/IP 四层/五层模型对比
2. **物理层**
   - 奈氏准则、香农公式、编码与调制
3. **数据链路层**
   - 封装成帧、透明传输、差错检测（CRC）
   - 介质访问控制：CSMA/CD、CSMA/CA
   - 以太网帧格式、MAC 地址、交换机工作原理
4. **网络层**
   - IP 编址、子网划分、CIDR
   - ARP、ICMP、DHCP、NAT
   - 路由算法：RIP、OSPF、BGP
   - IP 分组转发过程
5. **传输层**
   - UDP 与 TCP 对比
   - TCP 可靠传输：序号、确认、重传
   - TCP 流量控制（滑动窗口）
   - TCP 拥塞控制（慢开始、拥塞避免、快重传、快恢复）
   - 三次握手、四次挥手
6. **应用层**
   - DNS、HTTP、FTP、SMTP/POP3
   - Socket 编程基础
7. **网络安全**
   - 对称/非对称加密、数字签名、CA、HTTPS
