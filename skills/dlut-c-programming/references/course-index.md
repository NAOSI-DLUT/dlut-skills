## 周次题型

- week1-week4：基础输入输出、单位转换、条件分支、日期、字符类型、简单计算器。
- week5-week7：循环、素数、斐波那契、最大公约数、数组、排序、查找、去重、筛法。
- week8-week9：递归、矩阵、字符串转换、杨辉三角、回文、冒泡排序。
- week10-week11：字符串函数模拟、矩阵转置、成绩/姓名排序、结构体、点和链表。

## 使用方式

- 根据文件名可快速识别题型，例如 `prime`、`palindrome`、`sort`、`matrix`、`date`。
- 用户贴代码时先定位题型，再按输入、处理、输出三段检查。
