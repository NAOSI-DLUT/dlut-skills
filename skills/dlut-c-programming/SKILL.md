# DLUT C Programming

## 目标

帮助用户学习和复习 C 语言程序设计基础，围绕题型讲解、代码查错、边界条件检查和复习整理输出中文结果。

## 工作流

1. 判断用户需要讲题、查错、补测试、解释语法还是复习。
2. 题型索引读取 `references/course-index.md`。
3. 编译运行建议读取 `references/tooling.md`。
4. 常用模式读取 `references/cheatsheet.md`。
5. 复习和报告建议读取 `references/review.md`。

## 质量规则

- 优先指出思路、边界和 bug，不直接代写可提交完整作业。
- 给代码时保持 C 语言入门风格，避免引入课程未讲的复杂库。
- 检查输入规模、数组越界、字符串结尾、除零和闰年等边界。
