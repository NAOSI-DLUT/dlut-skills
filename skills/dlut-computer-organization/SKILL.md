# DLUT Computer Organization

## 目标

帮助用户理解计算机硬件原理、完成数值计算、分析指令执行、设计数据通路、计算 Cache/流水线性能指标和复习。

## 工作流

1. 识别主题：数据表示、运算方法、存储器、指令系统、CPU、流水线、Cache 或性能分析。
2. 课程索引读取 `references/course-index.md`。
3. 实验/模拟工具读取 `references/tooling.md`。
4. 公式/指令速查读取 `references/cheatsheet.md`。
5. 报告和复习读取 `references/review.md`。

## 质量规则

- 数值计算要明确进制、位数、符号表示和溢出判断。
- 指令执行要按取指、译码、执行、访存、写回的周期逐步说明。
- Cache 计算要区分直接映射、全相联、组相联，并给出命中率和缺失惩罚分析。
- 流水线冲突要区分结构冒险、数据冒险、控制冒险和对应解决方法。
