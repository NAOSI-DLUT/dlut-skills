# 计算机组成原理课程索引

## 章节与知识点

1. **计算机系统概述**
   - 计算机层次结构、性能指标（CPI、MIPS、MFLOPS）
   - Amdahl 定律
2. **数据的表示和运算**
   - 数制与编码：原码、反码、补码、移码
   - 定点数与浮点数（IEEE 754）
   - 算术逻辑单元（ALU）、加法器、乘法器
   - 浮点运算步骤
3. **存储器层次结构**
   - SRAM、DRAM、ROM 特性对比
   - 主存与 CPU 连接、多体交叉存储
   - Cache：映射方式（直接/全相联/组相联）、替换策略、写策略
   - 虚拟存储器：页式、段式、段页式、TLB
4. **指令系统**
   - 指令格式：操作码、地址码、寻址方式
   - CISC vs RISC
   - MIPS 指令格式（R/I/J 型）
5. **中央处理器（CPU）**
   - 数据通路：单周期、多周期
   - 控制器：硬布线、微程序
   - 指令流水线：冒险（结构/数据/控制）与处理
6. **总线与 I/O**
   - 总线分类、仲裁、定时
   - I/O 方式：程序查询、中断、DMA、通道
