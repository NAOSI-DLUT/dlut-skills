# 计算机组成原理速查

## 数据表示速查
- 补码范围：n 位有符号整数，范围 [-2^(n-1), 2^(n-1)-1]
- IEEE 754 单精度：1 位符号 + 8 位阶码（移码 127）+ 23 位尾数
- 浮点数规格化：尾数最高位为 1（隐含）

## 性能计算
- CPU 执行时间 = 指令数 × CPI × 时钟周期
- MIPS = 指令数 / (执行时间 × 10⁶) = 时钟频率 / (CPI × 10⁶)
- Amdahl 定律：加速比 = 1 / [(1-F) + F/S]

## Cache 计算
- 直接映射：行号 = 主存块号 mod Cache 行数
- 组相联：组号 = 主存块号 mod 组数
- 命中率 = 命中次数 / 总访问次数
- 平均访问时间 = 命中时间 + 缺失率 × 缺失惩罚

## MIPS 指令格式
| 类型 | 字段 | 用途 |
|------|------|------|
| R 型 | op(6) rs(5) rt(5) rd(5) shamt(5) funct(6) | 寄存器运算 |
| I 型 | op(6) rs(5) rt(5) immediate(16) | 立即数/Load/Store/Branch |
| J 型 | op(6) address(26) | 跳转 |
