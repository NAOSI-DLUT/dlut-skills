# 计算机组成原理实验工具

## Logisim / Logisim-evolution
- 数字电路仿真，用于设计 ALU、数据通路、CPU
- 官网：http://www.cburch.com/logisim/

## MIPS 模拟器
- MARS (MIPS Assembler and Runtime Simulator)
- 用于编写、汇编和运行 MIPS 汇编代码

## Verilog/VHDL（进阶）
- Vivado、Quartus、ModelSim
- 用于硬件描述和 FPGA 验证
