# 计算机组成原理复习要点

## 核心概念对比
| 概念 | 要点 |
|------|------|
| SRAM vs DRAM | 快/贵/稳 vs 慢/廉/需刷新；Cache 用 SRAM，主存用 DRAM |
| 直接映射 vs 组相联 | 实现简单/易冲突 vs 灵活/复杂度中等 |
| 硬布线 vs 微程序 | 速度快/难修改 vs 速度慢/易扩展 |
| CISC vs RISC | 指令复杂/变长/微程序 vs 指令简单/定长/硬布线 |

## 易错点
- 补码中 0 唯一，移码用于浮点数阶码便于比较大小
- 流水线加速比 ≠ 流水线级数（存在填充和排空）
- 数据冒险中的 RAW（写后读）最常见
- TLB 是页表的高速缓存，不是主存的 Cache

## 大题常见类型
1. 浮点数 IEEE 754 编码与运算
2. Cache 命中率与平均访问时间计算
3. 指令流水线时空图与冒险分析
4. 数据通路设计或单周期 CPU 控制信号分析
5. 磁盘 I/O 时间计算（寻道 + 旋转 + 传输）
