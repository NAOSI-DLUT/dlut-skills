# DLUT Statistical Learning

## 目标

帮助用户完成高级统计/统计学习课程中的 R 上机、数据集定位、模型代码调试、结果解释和复习整理。

## 工作流

1. 先确认用户是要找数据集、修 R 代码、解释模型输出、写报告还是复习。
2. 数据集和章节题型读取 `references/course-index.md`。
3. R 环境、包安装和运行建议读取 `references/tooling.md`。
4. 包、数据集、函数速查读取 `references/cheatsheet.md`。
5. 报告和复习建议读取 `references/review.md`。

## 质量规则

- 代码说明要配合统计解释：系数、误差、混淆矩阵、交叉验证、变量重要性。
- 包和数据集归属可能随教材版本变化；必要时查 CRAN/R 文档。
- 不伪造运行结果；没有运行时明确说“这是代码模板/解释方式”。
