# 软件工程速查

## UML 图速查
| 图类型 | 用途 | 核心元素 |
|--------|------|----------|
| 用例图 | 需求分析 | 参与者、用例、系统边界、关系 |
| 类图 | 静态结构 | 类名、属性、方法、关联、继承、依赖 |
| 时序图 | 动态交互 | 对象、生命线、消息、激活条 |
| 状态图 | 对象状态 | 状态、事件、转换、初始/终止 |
| 活动图 | 流程控制 | 活动、分支、合并、泳道 |
| 组件图 | 系统架构 | 组件、接口、依赖 |
| 部署图 | 物理部署 | 节点、构件、通信路径 |

## 设计模式速查
| 模式 | 类型 | 场景 |
|------|------|------|
| 单例 | 创建型 | 全局唯一实例（配置、连接池） |
| 工厂方法 | 创建型 | 创建对象时解耦具体类 |
| 观察者 | 行为型 | 事件订阅/通知机制 |
| 策略 | 行为型 | 封装算法族，互相替换 |
| 装饰器 | 结构型 | 动态添加功能 |
| 代理 | 结构型 | 控制对象访问 |

## 耦合度排序（高→低）
内容耦合 > 公共耦合 > 外部耦合 > 控制耦合 > 标记耦合 > 数据耦合 > 无直接耦合

## 内聚度排序（低→高）
偶然内聚 < 逻辑内聚 < 时间内聚 < 过程内聚 < 通信内聚 < 顺序内聚 < 功能内聚
