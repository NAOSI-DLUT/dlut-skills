# 软件工程工具

## UML 建模工具
- StarUML：轻量、易用
- PlantUML：文本生成 UML 图
- draw.io / diagrams.net：免费在线画图
- Visual Paradigm：功能全面

## 项目管理
- Jira：敏捷项目管理
- Trello：看板式任务管理
- Microsoft Project：甘特图、关键路径

## 版本控制
- Git + GitHub/GitLab/Gitee
- 分支策略：Git Flow、GitHub Flow

## 测试工具
- JUnit：Java 单元测试
- Selenium：Web 自动化测试
- Postman：API 测试

## 文档工具
- Markdown：轻量文档
- Swagger/OpenAPI：API 文档
- LaTeX：学术论文
