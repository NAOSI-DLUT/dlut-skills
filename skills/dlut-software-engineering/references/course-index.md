# 软件工程课程索引

## 章节与知识点

1. **软件工程概述**
   - 软件危机、软件工程定义
   - 软件生命周期：瀑布、增量、螺旋、敏捷
   - 敏捷开发：Scrum、Kanban、XP
2. **需求工程**
   - 需求获取、分析、规格说明、验证
   - 用例图、用例描述
   - 需求变更管理
3. **系统设计**
   - 总体设计：模块划分、耦合与内聚
   - 详细设计：流程图、N-S 图、PAD 图、PDL
   - 设计原则：SOLID、DRY、KISS
4. **UML 建模**
   - 用例图、类图、对象图
   - 时序图、协作图、状态图、活动图
   - 组件图、部署图
5. **设计模式**
   - 创建型：单例、工厂、抽象工厂、建造者、原型
   - 结构型：适配器、装饰器、代理、外观、桥接、组合、享元
   - 行为型：观察者、策略、模板方法、命令、迭代器、责任链、状态、访问者、中介者、备忘录、解释器
6. **软件测试**
   - 测试级别：单元、集成、系统、验收
   - 测试方法：黑盒（等价类、边界值、决策表）、白盒（语句、分支、路径覆盖）
   - 回归测试、自动化测试
7. **软件项目管理**
   - 进度管理：甘特图、PERT、关键路径
   - 成本估算：COCOMO、功能点
   - 风险管理、质量管理、配置管理
8. **软件维护**
   - 改正性、适应性、完善性、预防性维护
   - 可维护性度量
