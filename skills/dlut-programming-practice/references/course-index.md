# 项目索引

## 项目

- Clock：`GetLocalTime` 获取时间，EasyX 绘制时钟。
- Canvas：鼠标左右键绘图，Ctrl 组合改变图形尺寸，键盘清屏/换色/退出。
- Pinball：固定帧率、球反弹、挡板控制、失败条件。
- Miner2D：随机地形、插值、贴图、音频资源。
- Miner：软光栅 WIP，线性代数库、矩阵变换、逆双线插值贴图。

## 使用方式

- 先确认项目能打开和运行，再改交互或逻辑。
- 对图片和音频资源，优先检查相对路径和工作目录。
