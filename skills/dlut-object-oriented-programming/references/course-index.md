# 面向对象方法程序设计及实训课程索引

## 章节与知识点

1. **面向对象基础**
   - 面向对象 vs 面向过程
   - 对象、类、消息、接口
   - 封装、继承、多态
   - 抽象、模块化、信息隐藏
2. **面向对象分析（OOA）**
   - 需求获取与用例建模
   - 识别对象与类：名词分析法
   - 识别关系：关联、聚合、组合、继承、依赖
   - 建立领域模型
3. **面向对象设计（OOD）**
   - 架构设计：分层、MVC、MVVM
   - 详细设计：类设计、方法设计、接口设计
   - 设计原则：单一职责、开闭原则、里氏替换、接口隔离、依赖倒置
4. **UML 建模**
   - 用例图、类图、对象图
   - 时序图、通信图、状态图、活动图
   - 组件图、部署图
5. **设计模式**
   - 创建型：单例、工厂方法、抽象工厂、建造者、原型
   - 结构型：适配器、桥接、组合、装饰、外观、享元、代理
   - 行为型：责任链、命令、解释器、迭代器、中介者、备忘录、观察者、状态、策略、模板方法、访问者
6. **代码重构**
   - 坏味道识别：重复代码、过长方法、过大类、过长参数列表
   - 重构手法：提取方法、提取类、内联、移动方法
   - 重构与测试：先写测试再重构
7. **项目实训**
   - 团队协作与版本控制
   - 需求文档、设计文档、测试文档
   - 代码评审与持续集成
