# 面向对象方法程序设计及实训工具

## UML 建模工具
- StarUML：轻量易用，支持多种 UML 图
- PlantUML：文本生成 UML，便于版本控制
- draw.io / diagrams.net：免费在线画图
- Visual Paradigm：功能全面，支持正向/逆向工程

## IDE
- IntelliJ IDEA：Java 开发首选
- Eclipse：经典开源 IDE
- Visual Studio：C++ 开发
- VS Code：配合 Java/C++ 插件

## 版本控制
- Git + GitHub/GitLab
- 团队协作：分支策略、Pull Request、Code Review

## 测试框架
- JUnit：Java 单元测试
- Google Test：C++ 单元测试
- 测试驱动开发（TDD）：先写测试再写实现

## 代码质量
- SonarQube：代码质量分析
- Checkstyle/PMD：Java 代码规范检查
- ESLint：JavaScript/TypeScript 代码检查
