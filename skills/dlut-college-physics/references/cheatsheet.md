# 大学物理速查

## 常用常数
- 真空光速 c ≈ 3.00 × 10⁸ m/s
- 普朗克常量 h ≈ 6.63 × 10⁻³⁴ J·s
- 元电荷 e ≈ 1.60 × 10⁻¹⁹ C
- 真空介电常数 ε₀ ≈ 8.85 × 10⁻¹² C²/(N·m²)
- 真空磁导率 μ₀ = 4π × 10⁻⁷ T·m/A
- 玻尔兹曼常量 k ≈ 1.38 × 10⁻²³ J/K
- 阿伏伽德罗常量 Nₐ ≈ 6.02 × 10²³ mol⁻¹

## 力学公式
- 匀变速直线运动：v = v₀ + at, x = v₀t + ½at², v² = v₀² + 2ax
- 圆周运动：aₙ = v²/r = ω²r
- 牛顿第二定律：F = ma
- 动量定理：FΔt = Δp
- 动能定理：W = ΔEₖ
- 角动量：L = r × p, L = Iω

## 电磁学公式
- 库仑力：F = kq₁q₂/r², k = 1/(4πε₀)
- 电场强度：E = F/q
- 电势：V = W/q
- 高斯定理：∮E·dA = Q/ε₀
- 安培环路定理：∮B·dl = μ₀I
- 法拉第定律：ε = -dΦ/dt
- 洛伦兹力：F = q(E + v × B)

## 热学公式
- 理想气体：pV = nRT = NkT
- 内能：E = (i/2)nRT
- 热力学第一定律：ΔE = Q - W
- 卡诺效率：η = 1 - T₂/T₁

## 相对论公式
- 时间膨胀：Δt = γΔt₀, γ = 1/√(1-v²/c²)
- 长度收缩：L = L₀/γ
- 质能方程：E = mc²
