# 大学物理复习要点

## 核心概念对比
| 概念 | 要点 |
|------|------|
| 标量 vs 矢量 | 标量只有大小；矢量有大小和方向，运算遵循平行四边形法则 |
| 保守力 vs 非保守力 | 保守力做功与路径无关，有对应势能；非保守力反之 |
| 电场强度 vs 电势 | 电场强度是矢量，描述力的性质；电势是标量，描述能的性质 |
| 干涉 vs 衍射 | 干涉是有限光束叠加；衍射是波遇到障碍物偏离直线传播 |

## 易错点
- 高斯定理中 E 的通量只与内部电荷有关，但 E 由内外电荷共同决定
- 安培环路定理中 B 的环流只与穿过环路的电流有关
- 热力学第一定律中 Q、W、ΔE 的符号规定（系统吸热 Q>0，对外做功 W>0）
- 相对论中同时的相对性：同一地点同时发生的事件在所有惯性系同时

## 大题常见类型
1. 质点/刚体运动学与动力学综合
2. 静电场分布计算（高斯定理、叠加原理）
3. 电磁感应与电路综合（动生/感生电动势）
4. 热力学循环过程分析与效率计算
5. 光的干涉/衍射光强分布与条纹位置
6. 相对论时空坐标变换
