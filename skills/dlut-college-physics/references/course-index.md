# 大学物理课程索引

## 章节与知识点

1. **力学**
   - 质点运动学：位矢、速度、加速度
   - 牛顿运动定律
   - 动量与角动量、冲量
   - 功与能：动能定理、势能、机械能守恒
   - 刚体定轴转动
2. **电磁学**
   - 静电场：库仑定律、电场强度、电势、高斯定理
   - 静电场中的导体与电介质
   - 稳恒磁场：毕奥-萨伐尔定律、安培环路定理
   - 磁力、磁介质
   - 电磁感应：法拉第定律、楞次定律、自感与互感
   - 麦克斯韦方程组
3. **热学**
   - 气体动理论：理想气体状态方程、压强与温度
   - 能量均分定理、理想气体内能
   - 热力学第一定律
   - 热力学第二定律、熵、卡诺循环
4. **光学**
   - 光的干涉：双缝、薄膜、迈克尔逊干涉仪
   - 光的衍射：单缝、光栅、圆孔
   - 光的偏振
5. **近代物理**
   - 狭义相对论：洛伦兹变换、时间膨胀、长度收缩
   - 量子物理：黑体辐射、光电效应、康普顿效应
   - 玻尔模型、波粒二象性、不确定关系
   - 薛定谔方程、氢原子
