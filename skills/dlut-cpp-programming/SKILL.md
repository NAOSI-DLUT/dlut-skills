# DLUT C++ Programming

## 目标

围绕 C++ 程序设计课程，帮助用户理解面向对象题型、检查代码、整理复习笔记和生成类似练习。

## 工作流

1. 判断题目属于基础语法、类设计、运算符重载、模板、继承多态还是综合题。
2. 课程题型读取 `references/course-index.md`。
3. 编译运行读取 `references/tooling.md`。
4. 语法和模式读取 `references/cheatsheet.md`。
5. 复习建议读取 `references/review.md`。

## 质量规则

- 保持课程风格，优先使用标准库基础能力。
- 检查资源管理、拷贝构造、析构、const 正确性和边界条件。
- 不直接替用户生成可提交完整作业。
