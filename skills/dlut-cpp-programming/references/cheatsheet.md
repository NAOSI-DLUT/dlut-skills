# C++ 速查

- 构造函数：初始化对象状态。
- 析构函数：释放资源。
- 拷贝构造/赋值：管理深拷贝。
- `const` 成员函数：不修改对象状态。
- `friend`：常用于二元运算符访问私有成员。
- `operator+`, `operator-`, `operator<<`, `operator>>`：常见重载。
- `template<typename T>`：模板。
- 继承：`public` 继承表达 is-a。
- 多态：基类虚函数 + 派生类 override。

## 常见坑

- 返回局部变量引用。
- 析构函数非虚导致基类指针删除派生对象异常。
- 自赋值没有处理。
- 头文件中重复定义非 inline 函数。
