# DLUT Compiler Technology

## 目标

帮助用户完成编译技术实验、调试语法分析程序、整理输出格式和复习核心概念。

## 工作流

1. 先区分词法、文法、预测分析、LR/SLR、表达式求值或报告。
2. 实验索引读取 `references/course-index.md`。
3. 工具链读取 `references/tooling.md`。
4. 算法速查读取 `references/cheatsheet.md`。
5. 报告和复习读取 `references/review.md`。

## 质量规则

- 输出要明确输入文法格式、终结符/非终结符约定和结束符。
- 不直接给完整提交答案；优先给算法框架、调试点和错误定位。
- 涉及 flex/bison 语法时查当前官方文档或本地工具版本。
