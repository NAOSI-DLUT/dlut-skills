# 编译技术速查

- 词法分析：正则规则、最长匹配、动作代码。
- 消除直接左递归：`A -> Aα | β` 改为 `A -> βA'`, `A' -> αA' | ε`。
- FIRST：串首可能出现的终结符集合。
- FOLLOW：非终结符后面可能出现的终结符集合。
- 递归下降：每个非终结符一个函数。
- LR 项目：带点产生式。
- SLR：LR(0) 项目集 + FOLLOW 集决定 reduce。
- RPN：中缀表达式转后缀，便于栈求值。

## 检查点

- 文法是否有左递归或左因子。
- 预测分析是否冲突。
- ACTION/GOTO 是否完整。
- 错误恢复是否能前进，避免死循环。
