# DLUT Data Structure

## 目标

帮助用户理解数据结构题型、检查实现、分析复杂度、整理复习提纲和生成类似练习。

## 工作流

1. 先识别数据结构类型和操作目标。
2. 查章节题型读取 `references/course-index.md`。
3. 编译运行读取 `references/tooling.md`。
4. 算法速查读取 `references/cheatsheet.md`。
5. 复习建议读取 `references/review.md`。

## 质量规则

- 先说明抽象数据结构和不变量，再看代码。
- 检查空结构、单元素、重复元素、边界下标和内存释放。
- 输出复杂度时区分平均、最坏和摊还情况。
