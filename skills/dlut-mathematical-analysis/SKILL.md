# DLUT Mathematical Analysis

## 目标

围绕大工高等数学/数学分析课程，帮助用户理解题型、整理 MATLAB 上机代码、检查计算过程、生成复习计划和实验报告结构。

## 工作流

1. 先确认用户要处理的是课程复习、MATLAB 上机、题目讲解、代码调试还是报告整理。
2. 需要课程索引时读取 `references/course-index.md`。
3. 需要 MATLAB 环境或 VS Code 插件建议时读取 `references/tooling.md`。
4. 需要函数速查时读取 `references/cheatsheet.md`。
5. 需要复习策略或报告结构时读取 `references/review.md`。

## 质量规则

- 可以讲解思路、检查代码、指出错误、生成类似练习；不要直接包装成可提交的完整作业答案。
- MATLAB API 细节和插件状态可能变化；涉及安装或函数语法时用 Context7/MathWorks 文档核验。
- 数值绘图题要提醒用户检查定义域、复数值、步长、坐标范围和图例。
